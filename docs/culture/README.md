# Cultura del proyecto

Este espacio preserva la forma de pensar que ha guiado al Institutional Market Engine.

No contiene reglas técnicas ni decisiones de implementación. Contiene algo más duradero: los principios humanos, científicos y colaborativos con los que deseamos que el proyecto continúe evolucionando.

La cultura del proyecto parte de una convicción sencilla:

> El código explica qué hace el sistema. La cultura explica por qué vale la pena seguir construyéndolo.
