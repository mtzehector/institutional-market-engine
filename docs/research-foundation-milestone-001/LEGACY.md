# Legado

Si estás leyendo este documento, quizá estés continuando una investigación iniciada años atrás.

No asumas que las hipótesis aquí contenidas son verdades definitivas.

Ponlas nuevamente a prueba.

Mejóralas.

Reemplázalas cuando encuentres evidencia más fuerte.

Documenta las hipótesis rechazadas y explica por qué dejaron de sostenerse.

Conserva las líneas base y evita comparar experimentos incompatibles.

No agregues complejidad únicamente porque una variable parece importante.

Exige evidencia.

El mayor fracaso de este proyecto no sería equivocarse.

Sería dejar de formular preguntas.

Si el mercado cambia, permite que la plataforma cambie con él, sin perder el rigor, la reproducibilidad ni la memoria de lo aprendido.

Haz que la plataforma aprenda más de lo que nosotros logramos enseñarle.

Ese será el verdadero éxito del proyecto.
